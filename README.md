Mono-Mobilis
============

Aplicação multiplataforma (iOS, Android, Windows Phone) para acesso aos fórums do Sistema Online de Aprendizagem Solar

Configurando o Ambiente de Desenvolvimento
---------------

1. Instale o Visual Studio 2010 ou mais recente.

2. Instale as [Ferramentas da Xamarin] (http://xamarin.com/download) para desenvolvimento.

3. Instale a [SDK] (http://developer.windowsphone.com/en-us/downloadsdk) mais recente do Windows Phone.

4. Importe a solution Mobilis.sln.
